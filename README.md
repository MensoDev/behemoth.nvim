# behemoth.nvim

### Install

```sh
git clone https://github.com/MensoDev/behemoth.nvim.git ~/.config/nvim/ && nvim
```

### Uninstall

```sh
rm -rf ~/.config/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.local/share/nvim
```
